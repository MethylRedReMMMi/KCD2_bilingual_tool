# 天国拯救2双语mod[自用]
解压之后拖到天国拯救2根目录中的Mods文件夹中，如果没有Mods文件夹就创建一个。

MOD添加了物品，书本的双语文本，并且调整了其他双语字幕的格式。
2代的翻译有的时候不是特别的好，例如Scapular with saint被翻译为肩衣，恶一，恶二，恶三骰子英文分别是Lu die, Ci die, Fer die，对应了Lucifer。本MOD意在让玩家感受原版文字的风味，纯纯自用。
